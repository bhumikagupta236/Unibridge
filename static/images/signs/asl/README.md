# ASL Alphabet Images

## How to Add Real ASL Hand Sign Images

To replace the placeholder with actual ASL hand sign images:

### Option 1: Download from Free Sources

1. Visit these websites that offer free ASL alphabet images:
   - https://www.startasl.com/american-sign-language-alphabet/
   - https://www.gallaudet.edu/asl-resources/alphabet
   - Search Google Images for "ASL alphabet" with usage rights filter

2. Download images for each letter (A-Z)

3. Save them in this directory with these exact names:
   - A.jpg
   - B.jpg
   - C.jpg
   - ... (and so on through Z.jpg)

### Option 2: Take Your Own Photos

1. Use the descriptions in the learn-sign page as a guide
2. Take photos of your own hands forming each letter
3. Save with the same naming convention (A.jpg, B.jpg, etc.)

### Option 3: Use Free Stock Images

Search for "ASL alphabet hand signs" on:
- Unsplash.com
- Pexels.com  
- Pixabay.com

### Image Requirements

- Format: JPG, PNG, or GIF
- Recommended size: 400x400 to 800x800 pixels
- Name format: Single uppercase letter + .jpg (e.g., A.jpg, B.jpg)
- Background: Any (preferably plain/simple)

### After Adding Images

1. Refresh the page (Ctrl+Shift+R)
2. The real images will automatically replace the placeholders
3. Navigate through all 26 letters to verify

---

**Note:** The current setup looks for `.jpg` files. If you use `.png` or `.gif`, update the file extension in the JavaScript code in `templates/learn-sign.html` (line 124).
